METHOD 1: MANUAL INSTALLATION (RECOMMENDED - 5 minutes)
===============================================================================

Step 1: Download Poppler
-------------------------
Copy and paste this link in your browser:
https://github.com/oschwartz10612/poppler-windows/releases/download/v24.08.0-0/Release-24.08.0-0.zip

Save the ZIP file to your Downloads folder.

Step 2: Extract Files
---------------------
1. Right-click on the downloaded ZIP file
2. Click "Extract All..."
3. Extract to: C:\poppler
   (Create the folder C:\poppler if it doesn't exist)

Step 3: Find the BIN folder
---------------------------
After extraction, look for the bin folder. It's usually at:
C:\poppler\poppler-24.08.0\Library\bin

Copy this exact path!

Step 4: Update poppler_config.py
---------------------------------
Open the file: C:\Users\nk\paperai\answer_evaluation_app\poppler_config.py

Change this line:
POPPLER_PATH = r"C:\poppler\poppler-24.08.0\Library\bin"

To match YOUR actual bin folder location.

Step 5: Restart your app
------------------------
Close the Flask app and run again:
python app.py

‚úÖ DONE! Your app should work now!

===============================================================================
METHOD 2: ADD TO SYSTEM PATH (PERMANENT SOLUTION)
===============================================================================

If you want poppler available system-wide:

Step 1-3: Same as Method 1 (Download and extract)

Step 4: Add to System PATH
---------------------------
1. Press Windows Key + R
2. Type: sysdm.cpl
3. Press Enter
4. Click "Advanced" tab
5. Click "Environment Variables"
6. Under "System variables", find "Path"
7. Click "Edit"
8. Click "New"
9. Paste: C:\poppler\poppler-24.08.0\Library\bin
10. Click OK on all windows

Step 5: Restart Terminal
-------------------------
CLOSE all command prompt windows
Open a NEW command prompt

Step 6: Verify
--------------
pdfinfo -v

You should see Poppler version info!

Step 7: Update config
--------------------
In poppler_config.py, set:
POPPLER_PATH = None

(Because now it's in system PATH)
