===============================================================================
                    üêõ TROUBLESHOOTING
===============================================================================

Problem: Poppler not found after installation
Solution: 
1. Close ALL terminal windows
2. Open a NEW terminal
3. Try: pdfinfo -v
4. If still not working, manually add to PATH:
   C:\poppler\Library\bin

Problem: PyTorch installation fails
Solution:
pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu121

Problem: Out of memory error with DeepSeek-OCR
Solution:
- System will automatically fallback to Tesseract
- Or reduce image DPI in config.py (change OCR_DPI = 200)

Problem: ChromaDB errors
Solution:
python reset_db.bat

Problem: General dependency errors
Solution:
python -m pip install --upgrade pip
pip install -r requirements.txt --force-reinstall
